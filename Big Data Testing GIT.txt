RESTAssured:
given().  -> Request specifications
    baseURI
    headers
    queryparam
    pathParam
    body
    multipart(filepath)
    log().all()
when().  -> HTTP actions
    post()
    get()
    put()
    delete()
    patch()
then().   -> Response specifications
    log().all()
    statusCode()
    body(Matchers assertions)
    time()

For handling multiple windows:
* getWindowHandle(): For currently active window's handle as a String
* getWindowHandles(): For all windows' handles as a Set<String>
